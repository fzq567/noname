### 1.3 图片引用格式（原文档，作为图片引用概览）

extension.js 中使用的是无名杀扩展资源路径格式：

```javascript
// 武将图片
["ext:蒸蒸日上/ex_buqu.jpg", ...]

// 阵亡音频
["die:ext:蒸蒸日上/audio/die/ex_buqu.mp3", ...]

// 技能音频
audio: "ext:蒸蒸日上:2"
```

**关键发现**：`ext:蒸蒸日上/` 表示扩展根目录，图片迁移到子目录后，需要改为 `ext:蒸蒸日上/image/character/ex_xxx.jpg`
